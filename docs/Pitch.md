# Our Pitch
## What
We are creating a Bayesian-updated file structure.

## Why
This will allow for spatial-temporal data to be updated with limited readings. Its applications include Traffic pattern analysis and game analysis, where there are constant changes in position, without continuous readings.

P(A|B) = [P(B|A) * P(A)] / P(B) 
P(A) = The location we have is correct
P(B) = The new data point is correct.
